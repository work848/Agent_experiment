# AgentOS: Multi-Agent Orchestration System

AgentOS is an advanced, autonomous multi-agent backend engine designed to automate complex software engineering tasks, requirement analysis, and workflow execution. By leveraging multi-agent orchestration and Retrieval-Augmented Generation (RAG), it transforms high-level user intent into executable plans and verified code.

## Key Features

- **Autonomous Planning**: Automatically breaks down complex requirements into structured execution steps (MT-series tasks).
- **Multi-Agent Collaboration**: Orchestrates specialized agents (Planner, Coder, Tester, and Tool nodes) to handle different stages of the development lifecycle.
- **Visual Workflow Canvas**: Real-time visualization of task dependencies and execution flow.
- **Harness Workspace**: A comprehensive dashboard for monitoring agent intent, conversation history, and execution evidence.
- **Persistence & State Management**: Seamlessly reloads sessions and maintains persistent state across complex long-running tasks.
